# 互動式股票產業地圖

- **創立根產業**：點工具列「創立根產業」建立第一個產業（如產業A）。
- **新增相關產業**：點擊任一產業節點 → 選「新增相關產業」建立關聯產業（如產業B），節點間會顯示連線。
- **新增股票**：點擊產業 → 「在此產業下新增股票」，或在該產業下方輸入代碼後按「加入」，可加入多筆股票代碼。
- **儲存**：資料自動存於瀏覽器 localStorage，下次開啟可繼續編輯。

直接以瀏覽器開啟 `index.html` 即可使用。
